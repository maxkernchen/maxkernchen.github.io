WifiRanger

WifiRanger is a C# WPF project which calculates the approximate percentage of WiFi coverage for a given router and floor plan. It uses Walmart JSON Web Services to retrieve up to date prices and store links for a current list of 10 popular routers.

WiFi coverage is calculate by a modified free-space path loss formula which adds the natural log of the router's power to it. Some basic real world testing using simple Android applications proved this method to be fairly accurate. Below is the current formula WifiRanger uses to calculate the distance, in one direction, covered in meters.

$$m = 10^{\frac{(27.55 - (20 \cdot \log(frequencyMHz) + |dBm| +\ln(power)))}{20}}$$

Frequency is between 2400 - 5800 megahertz for most routers, dBm or decibel-milliwatts is usually -57 or -58 which is the lowest Wifi strength for reliable packet delivery. Power is different for every router, most of the time higher power = higher range.

Spot My Mood

Spot My Mood is an Android application which allows users to take a self image and display current emotions. These emotions are then connected to a Spotify playlist to best match them.

Spot My Mood is powered by Microsoft's Emotion API

An image is turned into a bitmap which is sent to Microsoft's Emotion Web Service. Then results are returned in an ArrayList which contains 8 emotions that are ranked from 0 to 1. The higher the number the more the emotion is seen.

These results are then converted into a visual graph using MPAndroidChart Below are PlayStore links and Spot My Mood's GitHub

Lynx

Lynx is a Peer-to-Peer file synchronization application written using Go. Using a shared meta file any machine running Lynx can receive and send files to other machines running Lynx.

Lynx implements a browser based GUI, which allows for portability between different operating systems and the use of modern web design components like Bootstrap and JQueryUI.

Lynx was part of a capstone project that all students must complete before graduating. This project was a partner project. My main responsibilities were the design and functionality of the UI and file parsing into structs and meta files. Currently I am making enhancements to all parts of Lynx which can be viewed at its GitHub linked below. Functionalities include tracking of player turns and usernames, prevention of incorrect moves or turns, game resets when a player disconnects, broadcast of current SOS board to all players. Written in Java and implements observer pattern.
